# Contribution guideline

- The editor **VS Code** is preferred.
- **C/C++ for Visual Studio Code** is recommanded to install with VS Code. It provides syntax checking and source code formatting.
- **Format the source code before saving.**
  - Use default formatting options.
- **End-of-lines are required in every source code files.**
- **Commit code with comments, please.** This is a collaborative project, others' codes are usually the hardest languages to understand.
  - **Put comments in header files at least.** 
- **This project should be a cross-platform project.** Be sure to code with care.
